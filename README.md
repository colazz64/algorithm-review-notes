# Algorithm Review Notes

This repository is my personal algorithm review notebook.

It records my LeetCode / NeetCode practice, mistake analysis, Java and Python solutions, topic-based templates, and Java-to-Python syntax notes.

## Why I Built This

I use this repository to turn daily algorithm practice into reusable review notes.  
The focus is not only on accepted solutions, but also on why I made mistakes, how I fixed them, and how I can avoid the same mistakes later.

## Current Focus

- Python syntax and common APIs
- Java to Python transition
- HashMap / dict and HashSet / set
- Sliding window and two pointers
- Basic recursion and tree problems
- NeetCode 150 and topic-based LeetCode practice

## Folders

| Folder | Purpose |
|---|---|
| `problems/` | One note per problem. Includes mistakes, idea, code, complexity, and review notes. |
| `topics/` | Pattern summaries such as HashMap, sliding window, binary search, recursion, and DP. |
| `syntax/` | Java to Python syntax comparison and common Python mistakes. |
| `code/` | Clean Java / Python solution files. |
| `assets/images/` | Screenshots and diagrams used in notes. |
| `templates/` | Reusable templates for future notes. |
